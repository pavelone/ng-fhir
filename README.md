## Development
`nodejs` is required for build.
We recommend install it using [nvm](https://github.com/creationix/nvm/blob/master/README.markdown)

```
git clone https://github.com/fhirbase/ng-fhir
cd ng-fhir
npm install
`npm bin`/grunt
```
